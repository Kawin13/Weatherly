# Weatherly — Weather Dashboard

A real-time weather dashboard built with HTML, CSS, and  JavaScript. No API key needed.

---

## What it does

- Search any city and get live weather data
- Shows temperature, humidity, wind speed, pressure, visibility, sunrise & sunset
- Switch between °C and °F
- Dark and light theme
- Detects your location automatically
- Remembers your last 5 searches
- Auto-refreshes every 5 minutes

---

## How to run

Just open `index.html` in your browser. That's it.

Or use a local server:

```bash
python3 -m http.server 8080
```

Then go to `http://localhost:8080`
